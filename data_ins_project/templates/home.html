Data Insighter is a web application that provides a variety of tools for extracting insights from data, such as images and text. The project was created as a way to showcase my back-end and data science skills and to build a community of data lovers.

The tech stack for Data Insighter includes:

  • Backend:
    • Django+Django REST Framework
  • Databases:
    • PostgreSQL
  • Data science:
    • NumPy
    • Pandas
    • Plotly
    • SciKit-Image
    • Pillow
    • Statistics
  • Frontend:
    • Bootstrap

The repository for Data Insighter is available on GitHub. Feel free to contribute to the project!
